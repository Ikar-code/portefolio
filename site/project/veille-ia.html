Veille IA — Application de veille technologique automatisée

Par Lucas Rajany · 8 avril 2026

Présentation du projet

J'ai conçu et développé Veille IA, une application web complète permettant de réaliser des veilles technologiques automatisées, principalement dans le domaine de l'intelligence artificielle.

L'objectif : supprimer le travail manuel de recherche et de synthèse. L'app recherche, score, résume et publie automatiquement — sans intervention humaine après configuration.

Fonctionnalités principales

1. Recherche multi-sources automatisée

  • Saisie de sujets multiples (ex : cybersécurité IA, LLM, deepfake…)
  • Recherche simultanée via DuckDuckGo et flux RSS spécialisés IA
  • Déduplication sémantique des résultats pour éviter les doublons

2. Scoring et filtrage intelligent

  • Chaque article reçoit un score basé sur la pertinence, la langue, la fiabilité de la source et la fraîcheur
  • Filtrage automatique des sources gouvernementales, low-quality et hors-sujet
  • Classement des contenus les plus pertinents en tête

3. Résumés IA par article

  • Analyse du contenu de chaque page
  • Génération de 3 à 5 points clés en français via Groq API (LLaMA 3)
  • Synthèse globale structurée par thème (menaces, outils, tendances…)

4. Publication automatique

  • Mise à jour d'une page WordPress via l'API REST
  • Upload direct sur hébergement FTP en HTML statique
  • Fusion intelligente du contenu : les anciens sujets sont conservés, les nouveaux s'ajoutent

5. Historique et comparaison

  • Sauvegarde de toutes les sessions par utilisateur dans Supabase
  • Comparaison entre deux sessions : articles nouveaux, disparus, évolutions
  • Analyse comparative générée par IA
  • Import d'historiques depuis un fichier HTML existant

6. Veille automatique planifiée

  • Chaque utilisateur choisit ses sujets et son heure d'envoi
  • Un cron GitHub Actions tourne toutes les heures (gratuit)
  • Envoi automatique d'un email HTML récapitulatif — sans que l'utilisateur soit connecté

7. Personnalisation du thème de publication

  • Éditeur de thème visuel intégré (couleurs, police, taille…)
  • 4 presets : Catppuccin, Light, Nord, Dracula
  • Aperçu live avant publication

8. Gestion des utilisateurs et abonnements

  • Inscription / connexion via Supabase Auth
  • Plan gratuit : 1 recherche offerte
  • Plan payant à 2,99€/mois : recherches illimitées + veille auto + email quotidien
  • Paiement via Stripe

Ce que j'ai appris

  • Architecture multi-utilisateurs avec isolation complète des données
  • Gestion de la fusion de données sans écrasement (upsert par clé)
  • Orchestration d'un pipeline IA : recherche → scoring → résumés → publication
  • Déploiement d'un service complet avec authentification, paiement et automatisation
  • Contraintes RGPD pour un service en ligne (hébergement EU)

Améliorations prévues

  • Export PDF / Markdown des veilles
  • Sources personnalisées par l'utilisateur (ajout de flux RSS propres)
  • Notifications push navigateur
  • Dashboard analytique (tendances sur la durée, mots-clés émergents)
  • Google OAuth
