Le Fil du Lord

Système complet de création d'articles automatisé par intelligence artificielle. Le pipeline combine recherche d'actualité, vérification factuelle, génération rédactionnelle, contrôle qualité et publication automatique.

Node.js React TypeScript Gemini API Groq API Supabase GitHub Actions Vercel

Description

Le Fil du Lord est une plateforme média automatisée capable de transformer un sujet d'actualité en article publiable. Le système fonctionne grâce à plusieurs étapes IA spécialisées : recherche de sujets, validation des informations, détection de doublons, rédaction journalistique et contrôle qualité avant publication.

Pipeline IA

1 Cron GitHub Actions déclenche automatiquement le pipeline.
2 Gemini recherche des sujets récents grâce au grounding Google Search.
3 Analyse anti-doublons : comparaison des événements déjà traités.
4 Vérification factuelle du sujet avant rédaction.
5 Groq génère l'article journalistique à partir uniquement des faits validés.
6 Gemini évalue la qualité de l'article avec un score sur 10.
7 Si le score est insuffisant, le système change de sujet ou régénère.
8 Publication automatique dans Supabase puis affichage sur le site React.

Anti-doublons intelligent

Le système ne compare pas uniquement les titres. Chaque sujet est analysé selon trois éléments :

  • Acteurs : les personnes, entreprises ou organisations impliquées.
  • Projet / Objet : le produit, événement ou œuvre concernée.
  • Événement : l'action principale (annonce, sortie, lancement, partenariat...).
  • Une correspondance sur 2 éléments déclenche une vérification supplémentaire IA.

Agents IA utilisés

🔎 Recherche

Gemini avec recherche Google identifie les sujets récents, les sources disponibles et les informations importantes.

✅ Vérification

Un agent contrôle la réalité du sujet, les dates, acteurs et informations avant validation.

✍️ Rédaction

Groq génère un article journalistique structuré avec titre, angle éditorial et contenu.

⭐ Scoring

Gemini analyse la qualité finale : fiabilité, structure, précision et intérêt éditorial.

Architecture technique

⚙️ Backend

Node.js orchestre les appels IA, les retries, la génération et la publication.

🗄️ Base de données

Supabase stocke les sujets, articles, scores, logs et statuts du pipeline.

🌐 Frontend

React + TypeScript affiche les articles publiés et fournit une interface administrateur.

🚀 Déploiement

GitHub Actions automatise l'exécution, Vercel héberge l'application web.

Points forts

  • Pipeline complet automatisé de la recherche à la publication.
  • Validation factuelle avant génération d'article.
  • Système anti-doublons basé sur les événements et non seulement les titres.
  • Contrôle qualité IA avec score et boucle de correction.
  • Traçabilité complète grâce aux logs.
  • Architecture évolutive permettant l'ajout de nouveaux agents IA spécialisés.
