Le Fil du Lord

Système complet de média d'actualité francophone automatisé par IA. Le projet combine recherche d'informations, vérification factuelle, génération journalistique, contrôle qualité et publication automatique.

Node.js React TypeScript Gemini API Groq API Supabase GitHub Actions Vercel

Description

Le Fil du Lord est une plateforme capable de transformer automatiquement des sujets d'actualité en articles publiables. Le système utilise plusieurs étapes IA spécialisées afin de limiter les erreurs : recherche de sujets, validation des informations, détection des doublons, rédaction journalistique et évaluation avant publication.

Ce que le système fait

  • Recherche d'actualité : Gemini avec accès Google Search identifie des sujets récents et pertinents.
  • Vérification factuelle : les sujets sont analysés avant rédaction afin de confirmer les informations importantes.
  • Anti-doublons intelligent : le système compare les nouveaux sujets avec les articles existants dans Supabase.
  • Rédaction automatique : Groq génère un article structuré avec titre, angle éditorial et contenu.
  • Contrôle qualité : Gemini attribue un score sur 10 et refuse les articles insuffisants.
  • Retry automatique : si un article obtient un mauvais score, le pipeline tente un nouveau sujet ou une nouvelle génération.
  • Publication : les articles validés sont enregistrés dans Supabase puis affichés sur le site.

Pipeline d'exécution

1 GitHub Actions lance automatiquement le pipeline via cron.
2 Recherche de sujets avec Gemini et Google Search.
3 Vérification des doublons et stockage dans Supabase.
4 Validation du sujet et création du dossier journalistique.
5 Rédaction automatique de l'article.
6 Scoring qualité IA avant publication.
7 Publication automatique si le score est suffisant.

Agents IA utilisés

🔎 Agent recherche — Scraping + flux RSS

Le système récupère automatiquement les informations depuis différentes sources grâce au scraping web et aux flux RSS. Les données brutes sont collectées puis transformées en sujets exploitables pour le pipeline éditorial.

✅ Agent vérification — Gemini 2.5 Flash Lite

Analyse les sujets collectés avant rédaction. Il vérifie la cohérence des informations, les dates, les acteurs concernés et les faits disponibles afin de limiter les erreurs et les informations inventées.

✍️ Agent rédaction — Qwen3-32B (Groq)

Transforme les informations vérifiées en article journalistique complet. Le modèle génère le titre, l'angle éditorial et le contenu final en respectant la ligne éditoriale du média Le Fil du Lord.

⭐ Agent qualité — Gemini 3.1 Flash Lite

Évalue chaque article avant publication avec un score sur 10. Il analyse la fiabilité, la structure, la qualité rédactionnelle et le respect des règles éditoriales avant validation.

Architecture technique

⚙️ Backend

Node.js orchestre les appels IA, la logique métier, les retries et la publication.

🗄️ Base de données

Supabase stocke sujets, articles, scores, statuts et journaux.

💻 Frontend

React + TypeScript affiche les articles et fournit une interface d'administration.

🚀 Déploiement

GitHub Actions automatise l'exécution et Vercel héberge l'application.

Points clés

  • Pipeline complet de la recherche à la publication.
  • Double contrôle IA avec vérification et scoring.
  • Système anti-doublons basé sur les événements.
  • Traçabilité complète grâce aux logs.
  • Architecture évolutive pour ajouter de nouveaux agents IA.
